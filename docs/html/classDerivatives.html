double Derivatives::Dxx_local ( const Field &  f,
size_t  i,
size_t  j,
size_t  k 
) const

Compute the local second derivative ∂²f/∂x² at a single grid point.

Evaluates the second–order central finite difference at coordinates (i,j,k): ( f(i+1,j,k) + f(i-1,j,k) − 2 f(i,j,k) ) / dx². Performs minimal boundary checking: if i is on the domain boundary, returns 0.0.

Parameters
fThe input scalar field.
iGrid index in the x–direction.
jGrid index in the y–direction.
kGrid index in the z–direction.
Returns
The second derivative ∂²f/∂x² at (i,j,k).
