\doxysubsection{Detailed Description}
Standalone test fixture for the \mbox{\hyperlink{classLinearSys}{Linear\+Sys}} class. 

This fixture does not inherit from any other test fixture. It creates all necessary objects (\mbox{\hyperlink{structGrid}{Grid}}, \mbox{\hyperlink{classField}{Field}}, \mbox{\hyperlink{classVectorField}{Vector\+Field}}) from scratch required to test the \mbox{\hyperlink{classLinearSys}{Linear\+Sys}} interface.

It is declared as a \textquotesingle{}friend\textquotesingle{} of \mbox{\hyperlink{classLinearSys}{Linear\+Sys}} to allow access to private members for testing.
